Covid 19 Live Stats is a WordPress Plugin to display Coronavirus (COVID-19) outbreak live updates in WordPress website. Display Covid-19 stats on your pages/posts/sidebars using simple widgets and shortcodes.

== Description ==

Covid-19 Live Stats WordPress plugin is an awesome plugin for displaying Coronavirus (COVID-19) data on your WordPress site. You can simply use the shortcodes and widgets to show live data on your posts, pages or sidebars.

== Data Source ==

All COVID-19 Live Data shown using this plugin is provided by Coronavirus monitor (https://rapidapi.com/astsiatsko/api/coronavirus-monitor/details) By astsiatsko (https://rapidapi.com/user/astsiatsko). You can check the Privacy Policy here (https://rapidapi.com/privacy-policy).

== Installation ==

To install follow this steps:

1. Upload `covid19livestats` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
